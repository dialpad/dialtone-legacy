Font Weight

Utilities to change an element's font-weight.

# Variables

Variable Output
var(--fw-normal) 400
var(--fw-bold) 700

# Classes

Class Output
.d-fw-normal font-weight: var(--fw-normal) !important;
.d-fw-bold font-weight: var(--fw-bold) !important;

# Usage

Use d-fw-{n} to change an element's font-weight.

.d-fw-normal

The quick brown fox jumps over the lazy dog.

.d-fw-bold

The quick brown fox jumps over the lazy dog.

<p class="d-fw-normal">...</p>
<p class="d-fw-bold">...</p>
