# Example

Use d-hmx{n}p or d-hmx{n} to set a maximum height percentage for an element. This can be combined with d-h{n}p and d-hmn{n} to have an element fill a certain height range.

1
<div class="d-h216">
    <div class="d-h72 d-hmx50p">1</div>
</div>
