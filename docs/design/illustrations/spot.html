{% header "h2", "Theming" %} {% paragraph %}Spot illustrations can use the current theme's primary color within the SVG. In order to do this you must set one of the following classes on your SVG path tags:{% endparagraph %}
  • d-svg-primary--stroke - to set the stroke to the theme's primary color
  • d-svg-primary--fill - to set the fill to the theme's primary color
{% paragraph %}We also recommend using opacity to create other colors derived from the theme's primary color where possible.{% endparagraph %} {% codeWell %} {% codeWellHeader %} In the below example we are using d-svg-primary--fill and d-svg-primary--stroke as a css class within our SVG. If you were to view the SVG locally, it would use the set color #3dffab. However once this is built into dialtone the local style tags would be removed and it would use the primary color of the dialtone theme instead. {% endcodeWellHeader %} {% codeWellFooter %} {% highlight html linenos %} {% endhighlight %} {% endcodeWellFooter %} {% endcodeWell %}
