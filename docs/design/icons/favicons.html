{% header "h2", "Sizes" %} {% paragraph %}It used to be that a 16x16 {% code %}favicon.ico{% endcode %} file was enough. With the explosion of smart devices the last 15 years, its a bit more confusing about what favicon to provide to which device. Compounding this problem is the seemingly lack of firm documentation about favicons and their sizes and formats. Below represents our attempt to compile a full list of favicons. Much thanks goes to the Favicon Cheat Sheet, RealFaviconGenerator.net and its author Philippe Bernard, and "The 2020 Guide to Favicons for Nearly Everyone and Every Browser" by Greg Gant.{% endparagraph %} {% paragraph %}Each project can choose which sizes they will support, but Dialtone's goal is to at least provide all the Favicon sizes. Legacy favicons icons are grayed out and can be skipped if desired.{% endparagraph %} {% for i in site.data.favicons.sizes %} {% assign size = i.size %} {% assign type = i.type %} {% assign desc = i.desc %} {% endfor %}
