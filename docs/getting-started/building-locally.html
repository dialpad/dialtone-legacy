--- layout: page title: Build Dialtone Locally description: A more thorough guide for installing Dialtone locally. ---
{% paragraph %}We're excited you want to install Dialtone locally as this most likely means you'll be contributing soon! Before you get to get started though, please make sure you've read our contributing docs.{% endparagraph %}
{% header "h2", "Install Node & npm" %}
{% paragraph %}To run Dialtone locally, you must have Node and NPM (Node Packet Manager) installed. Click here to download Node. The recommended Node version is fine. NPM is included with Node. If you already have Node installed, you may move onto the next step.{% endparagraph %} {% paragraph %}Once Node finishes installing, ensure it is installed properly by typing the following command in your Terminal window:{% endparagraph %} {% codeWell %} {% codeWellFooter %} {% highlight bash linenos %} node -v {% endhighlight %} {% endcodeWellFooter %} {% endcodeWell%}
{% paragraph %}You should see the following response:{% endparagraph %} {% codeWell %} {% codeWellFooter %} {% highlight bash linenos %} v14.16.1 {% endhighlight %} {% endcodeWellFooter %} {% endcodeWell%}
{% header "h2", "Clone Project" %}
{% paragraph %}Download the project:{% endparagraph %} {% codeWell %} {% codeWellFooter %} {% highlight bash linenos %} // SSH git clone git@github.com:dialpad/dialtone.git // HTTPS git clone https://github.com/dialpad/dialtone.git {% endhighlight %} {% endcodeWellFooter %} {% endcodeWell%}
{% paragraph %}Then {% code %}cd{% endcode %} into the Dialtone directory:{% endparagraph %} {% codeWell %} {% codeWellFooter %} {% highlight bash linenos %} cd ./path/to/dialtone {% endhighlight %} {% endcodeWellFooter %} {% endcodeWell%}
{% header "h2", "Install Dependencies" %}
{% paragraph %}Dialtone uses Gulp to automate its various workflows. Run the following command to install Gulp and all other project dependencies:{% endparagraph %} {% codeWell %} {% codeWellFooter %} {% highlight bash linenos %} npm install {% endhighlight %} {% endcodeWellFooter %} {% endcodeWell%}
{% header "h2", "Building Dialtone" %}
{% paragraph %}You're now ready to build Dialtone! To build and run the development server: {% endparagraph %} {% codeWell %} {% codeWellFooter %} {% highlight bash linenos %} npm start {% endhighlight %} {% endcodeWellFooter %} {% endcodeWell%}
{% paragraph %}Once finished, visit {% code %}http://localhost:4000/{% endcode %}.{% endparagraph %}
