{% header "h2", "Accessibility" %} {% paragraph %}The best accessibility is semantic HTML. Most screen readers understand how to parse inputs if they’re correctly formatted. When it comes to checkboxes, there are a few things to keep in mind:{% endparagraph %} {% ul %}
  • All inputs should have an {% code %}id{% endcode %} attribute.
  • Associate checkbox labels with their inputs using the {% code %}for{% endcode %} attribute. This correlates with the checkbox's {% code %}id{% endcode %}.
  • If you have a group of related checkboxes, use a {% code %}fieldset{% endcode %} to group them and a {% code %}legend{% endcode %} to title the group.
    • {% endul %} {% paragraph %}For further information, please visit Gov.UK’s article, "Using the fieldset and legend elements".{% endparagraph %}
