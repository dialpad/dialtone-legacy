{% header "h2", "Accessibility" %} {% paragraph %}The best accessibility is semantic HTML. Most screen readers understand how to parse buttons if they’re correctly formatted. When it comes to toggles, there are a few things to keep in mind:{% endparagraph %} {% ul %}
  • All toggle buttons should have an {% code %}id{% endcode %} attribute.
  • Associate toggle labels with their buttons using the {% code %}for{% endcode %} attribute. This correlates with the toggle's {% code %}id{% endcode %}.
  • If you have a group of related toggles, use a {% code %}fieldset{% endcode %} to group them and a {% code %}legend{% endcode %} to title the group. For further information, please visit Gov.UK’s article, "Using the fieldset and legend elements".
