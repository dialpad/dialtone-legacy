{% header "h2", "Hover" %}

Use {% code %}h:d-bc-{color}{% endcode %} to change an element's border color spot on {% code %}:hover{% endcode %}.

{% highlight html linenos %} {% endhighlight %}
{% header "h2", "Focus" %}

Use {% code %}f:d-bc-{color}{% endcode %} to change an element's border color when in {% code %}:focus{% endcode %} or {% code %}:focus-within{% endcode %} states.

{% highlight html linenos %} {% endhighlight %}
{% header "h2", "Dark Mode" %}

Use {% code %}d:d-bc-{color}{% endcode %} to set different border color when the user has dark mode enabled.
