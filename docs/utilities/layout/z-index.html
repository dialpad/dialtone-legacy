{% header "h2", "Variables" %}

When writing Less, you can set an element's z-index by using a variable ({% code %}@zi-{level}{% endcode %}). The table below lists the available variables, output, and a description for when they should be used.

{% for i in z-index %} {% assign name = i.name %} {% assign description = i.description %} {% assign output = i.output %} {% endfor %}
Variable Output Description
@zi-{{ name }} {{ output }} {{ description }}
{% header "h2", "Classes" %}

Set an element's z-index by using a class ({% code %}.d-zi-{level}{% endcode %}>). These classes match up with the variables names listed above. The table below lists the available z-index levels, class names, and the CSS output.

Class Output
