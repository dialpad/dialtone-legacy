Directions

By default, all .d-bgg-[color] classes have a gradient direction of right to left. You can change this though by applying a .d-bgd-[direction] class, which overrides the default CSS var.

{% for i in visual-styles.bgg-directions %} {% assign dir = i.direction %} {% assign output = i.output %} {% endfor %}
Class Output
.d-bgd-{{ dir }} --bgg-direction: {{ output }} !important;
{% for i in visual-styles.bgg-directions %} {% assign dir = i.direction %}
.d-bgg-ash .d-bgd-{{ dir }}
{% endfor %}

Percentage

By default, all colors in the .d-bgg-[color] classes are determined by lightening and darkening the core color by 7.5%. You can reduce or increase this percentage by using a .d-bgp-[sm|lg|xl] class, which overrides the default CSS var.

{% for i in visual-styles.bgg-percentage %} {% assign value = i.value %} {% assign class = i.class %} {% assign output = i.output %} {% endfor %}
Size Class Output
{{ value }} {% if class != "N/A" %}.d-bgp-{% endif %}{{ class }} --bgg-percentage: {{ output }} !important;
{% for i in visual-styles.bgg-percentage %} {% assign value = i.value %} {% assign class = i.class %}
.d-bgg-blue-light{% if class != "N/A" %} .d-bgp-{{ class }}{% endif %}
{% endfor %}

Opacity

In addition to the percentage and direction controls above, Dialtone also provides the ability for you to control one background color's opacity using the .d-bgo[#] classes.

{% for i in visual-styles.bgc-opacity %} {% assign stop = i.stop %}
.d-bgg-slate .d-bgp-sm .d-bgo{{ stop }}
{% endfor %}
