{% paragraph %}All font colors pass the WCAG 2.1 Level AA contrast ratio requirements (ratio >= 4.5:1) and most pass WCAG 2.1 Level AAA requirements (ratio >= 7:1). The contrast ratio value is noted with the colors below. Please use only these colors or variations of these colors which pass WCAG 2.1 Level AA contrast ratio requirements.{% endparagraph %}
